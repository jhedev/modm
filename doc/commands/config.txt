config: Show configuration for Modm or a module.
usage: config [<module>]

  Display the current configuration of Modm. This shows all environment
  variables that affect the execution of Modm, together with their current
  values.

  If a module <module> is specified, the configuration for that module is shown.
  This includes the contents of the module file, enabling the user to see how
  the enviroment changes when a module is loaded.
